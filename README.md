Music Playlist Manager (C)

A console-based music playlist manager written in **C**.  
It allows users to **add, display, reverse, shuffle, and rotate** songs in the playlist using arrays and stack data structure.

 Features
- Add songs to the playlist
- Display all songs
- Reverse playlist using **stack**
- Shuffle playlist (swap neighboring songs)
- Rotate playlist (move last song to first)
- Simple console interface

---

 Requirements
- Windows OS or any system with **C compiler**
- GCC / Dev-C++ / Code::Blocks / VS Code

---

 How to Run
1. Open terminal or your C IDE
2. Compile the program:
```bash
gcc playlist.c -o playlist
